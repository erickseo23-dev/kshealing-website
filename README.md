# KS Healing Website

Sitio web oficial de KS Healing - Energía Crística para la Transformación.

## Descripción

KS Healing es un sistema de sanación energética que utiliza la Energía Keiouvos Stharef para facilitar la transformación profunda de los sistemas energéticos humanos.

Este repositorio contiene el código fuente del sitio web oficial: [kshealing.com](https://kshealing.com)

## Estructura del Proyecto

- **client/** - Aplicación React frontend con Vite
- **server/** - Backend con Express.js
- **shared/** - Código compartido entre cliente y servidor
- **patches/** - Parches personalizados para dependencias

## Tecnologías

- **Frontend:** React 19, TypeScript, Vite, TailwindCSS, Radix UI
- **Backend:** Express.js, Node.js
- **Base de datos:** Supabase
- **Animaciones:** Framer Motion
- **Formularios:** React Hook Form
- **Validación:** Zod

## Instalación

```bash
# Instalar dependencias
pnpm install

# Ejecutar en desarrollo
pnpm dev

# Compilar para producción
pnpm build

# Iniciar servidor de producción
pnpm start
```
